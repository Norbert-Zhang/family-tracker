1. https://console.firebase.google.com/project/family-tracker-a92ae/overview

第一步：创建免费 Firebase 项目（5分钟）

打开 console.firebase.google.com，用 Google 账号登录
点击 "添加项目" → 起个名字（如family-tracker）→ 创建
进入项目 → 左侧菜单 构建 → Realtime Database → 点击"创建数据库"

选择离你最近的区域（如 新加坡）
选择 "以测试模式启动" → 完成


点击左上角 项目概览 → 点击 </> 图标 → 注册网页应用
复制 firebaseConfig = { ... } 花括号里面的全部内容

第二步：配置 APP

用浏览器打开下载的 HTML 文件
点击底部 ⚙️ Firebase 配置 → 粘贴刚复制的内容 → 保存
只需配置一次，之后自动记住

2. 把HTML文件上传到静态托管（例如Github)上，并部署发布
